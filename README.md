T20 World Cup: Player Impact & Match Insights

📌 Project Overview
Transforming complex, nested JSON data into a clean relational model to analyze player performances in the T20 World Cup. This project focuses on "Impact" rather than just "Runs."

🛠️ Tech Stack
- Data Parsing: JSON to Relational (Power Query)
- Modeling: Star Schema / Data Modeling in Power BI
- Advanced Analytics: DAX (Player Impact Index)

📊 Key Performance Metrics
- Data Optimization: Integrated multiple JSON sources into a unified model, improving data retrieval speed by 50%.
- Impact Index: Developed a "Player Impact Score" identifying undervalued players who had a 20% higher win-contribution than marquee names.
- Strategic Insights: Visualized a 10% variance in economy rates based on stadium boundary dimensions.

🏏 Dashboard Sections
- Batting Powerplay Analysis
- Bowling Death Over Efficiency
- Head-to-Head Team Comparison
